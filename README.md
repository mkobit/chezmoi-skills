Agent skills for use with chezmoi.

These skills follow the [agent skills specification](https://agentskills.io/specification)
so they can be used by any skills-compatible agent, including Claude Code and
Codex CLI.

Explore the `skills/` directory to discover available capabilities and
instructions.
