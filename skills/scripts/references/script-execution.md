# Script execution and naming

- [Understand how scripts work](https://www.chezmoi.io/user-guide/use-scripts-to-perform-actions/#understand-how-scripts-work): An overview of how chezmoi executes scripts, including the different types of script prefixes (`run_`, `run_once_`, `run_onchange_`).
- [The `.chezmoiscripts/` directory](https://www.chezmoi.io/reference/special-directories/chezmoiscripts/): Documentation on the special directory where chezmoi expects scripts to be located and how they are parsed.
- [Script environment variables](https://www.chezmoi.io/user-guide/use-scripts-to-perform-actions/#script-environment-variables): Information about the environment variables chezmoi sets when running scripts, like `CHEZMOI=1` and template data.
